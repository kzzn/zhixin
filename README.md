# 智信 (Zhixin) 即时通讯系统 - 功能清单

本文档详细梳理了海豚智信（Zhixin）即时通讯系统的核心功能模块，对比了 PC 端（基于 Tauri + Vue 3 / Naive UI）与移动端（基于 Vue 3 + Vant）在各项功能上的实际支持情况，并详细说明了 AI 助手（豚豚虾）的配置模型与 MCP 工具能力。

---

## 1. 登录与认证模块 (Authentication)

| 模块 | 功能 | 描述 | PC支持 | 移动端 |
| :--- | :--- | :--- | :--- | :--- |
| **登录与认证** | 账号密码登录 | 支持使用账号/手机号与密码进行身份验证登录。 | 支持 | 支持 |
| **登录与认证** | 验证码登录 / 找回密码 | 支持通过手机验证码进行快速登录或重置/找回密码。 | 支持 | 支持 |
| **登录与认证** | 二维码扫码登录 | 支持 PC 端展示登录二维码，移动端通过扫码并进行二次确认完成电脑端安全登录。 | 支持 (展示二维码) | 支持 (扫码及确认) |
| **登录与认证** | 服务器/网络代理配置 | 支持自定义代理服务器及 API 后端服务地址，以适应私网部署和特殊网络环境。 | **未启用/隐藏**<br>(代码中被 `v-if="false"` 屏蔽) | 不支持 |
| **登录与认证** | 多端设备登录管理 | 支持展示当前在线设备列表，允许远程将其他在线端强制下线，并在被挤下线时进行实时通知提醒。 | 支持 | 支持 |

## 8. AI 智能助手 “豚豚虾” 模块 (AI Assistant)

系统目前内置的 AI 智能助手为 **“豚豚虾”**。它拥有独立的模型供应商配置入口，并完全集成了丰富的 **MCP 工具箱 (Model Context Protocol)**，具备读取本地文件、修改代码、执行脚本指令以及操作 IM 消息与云盘的全面 Agent 能力。

| 模块 | 功能 | 描述 | PC支持 | 移动端 |
| :--- | :--- | :--- | :--- | :--- |
| **AI 智能助手** | 大模型供应商配置 | 提供独立的 `ModelConfigModal`，支持自主配置 Ollama、OpenAI 及自定义 AI 平台的 API Base 接口地址、API Key 密钥，并支持一键📡自动拉取或手动配置模型列表。 | 支持 | 支持 |
| **AI 智能助手** | 本地开发项目关联 | 允许将本地开发项目目录（Workspace）引入“豚豚虾”工作区，支持项目树展示、置顶和展开管理。双端公用一套项目存储逻辑。 | 支持 | 支持 |
| **AI 智能助手** | 任务执行计划跟踪 | 支持可视化配置 AI 规划任务，并通过监听 Rust 后台任务状态事件，实时渲染 AI 任务步骤与执行进度。 | 支持 | 支持 |
| **AI 智能助手** | 时光机历史对话管理 | 支持“新对话”、会话检索、重命名、归档及在切走空白对话时自动静默销毁临时会话。 | 支持 | 支持 |
| **AI 智能助手** | 智能体插件/工具管理 | 提供了 `PluginManagerModal` 管理“豚豚虾”底座插件，集成 30+ 种 MCP 工具，使其可代理操作客户端及物理设备。 | 支持 | 部分 (仅支持会话中触发工具) |

| **系统底层** | 系统相册与公共存储写入 | 支持将接收的媒体图片安全写入手机系统的原生系统相册，或将文档文件保存至公共共享存储区中。 | 支持 | 支持 |
| **系统底层** | 独立沙箱路径拷贝 | 为绕过移动端沙箱权限限制，提供了在底层将外部选取文件的原生 URI 异步拷贝到应用独立沙箱临时缓存目录的功能。 | 不支持 | 支持 |
